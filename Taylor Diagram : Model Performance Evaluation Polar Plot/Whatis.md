🎯 **模板标准命名**

泰勒图 / 模型性能多维极坐标评估图 (Taylor Diagram / Model Performance Evaluation Polar Plot)

🏷️ **AI 检索高频关键词 (Tags)**

建议将这些专业术语作为标签，涵盖其极特殊的数学几何映射特征：

- **基础图形**: 泰勒图, 极坐标评估图, Taylor diagram, polar statistical plot, model skill plot.
- **网格与拓扑**: 三重交叉网格 (triple overlapping grids), 单象限极坐标 (first-quadrant polar), 放射状辅助线 (radial lines), 偏心同心圆 (eccentric concentric circles).
- **视觉映射**: 观测参考基准点 (observation reference point/anchor), 标准差半径 (standard deviation radius), RMSD 距离 (RMSD distance), 相关系数角度 (correlation angle).
- **适用场景**: 气象/气候模式模拟评估 (climate model evaluation), 多算法/多模型预测性能联合对比.

📝 **核心特征描述 (Prompt 提示词模板)**

你可以直接复制这段描述作为该模板的 AI 提示词（Prompt）。这段话精准剥离了具体数据，向 AI 解释了泰勒图极为特殊的三维变量二维化映射逻辑：

“这是一个用于多模型或多变量性能评估的 **泰勒图 (Taylor Diagram)**。该图表基于单象限的变体极坐标系构建，在一个二维画布中同时映射三个统计学指标：1. 从坐标原点出发的**径向距离（半径）\**映射数据的 \*\*标准差 (Standard Deviation)\*\*；2. 极角的\**余弦值**映射模型与参考数据的 **相关系数 (Correlation)**，通常在外圆弧标注刻度；3. 图中各散点到X轴上特定的“参考/观测基准点”的**欧氏几何距离**映射 **均方根误差 (RMSD)**。全局视觉包含三套交叉网格：以原点为中心的同心圆弧（标准差网格）、从原点发散的放射线（相关系数网格）、以及以参考基准点为中心的虚线同心圆弧（RMSD 网格）。”

🗂️ **结构化特征解析 (模板属性卡片)**

| **核心组件**                         | **绘图类型与视觉特征**                                       |
| ------------------------------------ | ------------------------------------------------------------ |
| **整体坐标系**                       | **定制化极坐标变体**。完全去除了传统的笛卡尔直角坐标系（X/Y轴），而是通过底层计算将所有统计量强行转化为 `cos(theta)*R` 和 `sin(theta)*R` 的几何点。 |
| **参考基准点 (Reference Point)**     | **绝对锚点**。通常位于 X 轴（即相关系数为 1 的绝对水平线上），其半径等于观测数据的实际标准差。所有其他模型的优劣均通过与该点的距离和角度来直观判定。 |
| **多重网格系统 (Overlapping Grids)** | **三重交叉网格**。由底层线条（`plot` 函数）硬编码绘制：外圈弧线+放射状直线形成坐标系外框；不同颜色/线型的内部圆弧用于辅助读取 STD 和 RMSD 数值。 |
| **主体视觉 (Scatter & Text)**        | **散点与文字标注组合**。每个模型/算法作为一个散点绘制在图谱中，带有独立的颜色和形状标记（Marker），点旁边紧贴文本（Text）用于标明对应的算法名称或编号。 |
